# Kaizen IQ SEO Content War Room Playbook

## Purpose

This repo uses project-level Claude Code subagents to sharpen Kaizen IQ SEO content through a structured review board. Each agent applies a different form of pressure so the final website content is not merely optimized, but strategically superior.

## Agent Roles

| Agent | Purpose |
|---|---|
| search-intent-researcher | Defines buyer search demand, intent, funnel stage, and keyword clusters. |
| serp-competitor-intelligence | Maps competitor patterns, SERP conventions, and differentiation whitespace. |
| seo-content-specialist | Turns strategy into metadata, headings, briefs, FAQs, and on-page SEO recommendations. |
| eeat-trust-architect | Strengthens proof, credibility, methodology, responsible AI posture, and trust. |
| conversion-strategist | Sharpens CTAs, offer clarity, buyer objections, and lead-generation logic. |
| contrarian-red-team | Attacks weak content, generic language, unsupported claims, and publish-readiness. |
| technical-seo-architect | Audits metadata, schema, crawlability, headings, internal links, and technical SEO. |
| brand-authority-editor | Elevates the copy into premium Kaizen IQ strategy-consulting language. |
| analytics-learning-loop | Uses performance data to recommend refreshes and experiments. |

## Recommended Workflow

Use these agents in sequence when creating or refreshing a strategic page:

1. `search-intent-researcher`
2. `serp-competitor-intelligence`
3. `seo-content-specialist`
4. `eeat-trust-architect`
5. `conversion-strategist`
6. `contrarian-red-team`
7. `brand-authority-editor`
8. `technical-seo-architect`
9. `analytics-learning-loop` after publishing and gathering data

## Example Claude Code Prompt

Use the search-intent-researcher agent to analyze `services.html` for Kaizen IQ's data, AI, analytics, and marketing consulting services. Then use the serp-competitor-intelligence agent to identify the likely competitive gaps. Then use the seo-content-specialist agent to produce a revised SEO content brief. After that, use the contrarian-red-team agent to challenge the brief before any implementation.

## Publish Gate

Do not publish or merge SEO copy if any of these score below 4 out of 5:

- Authority
- Usefulness
- Conversion clarity
- Brand strength

## Page-Level Quality Standard

Every major page should answer:

1. What buyer problem does this page solve?
2. What exact search intent does it satisfy?
3. Why should Kaizen IQ be trusted?
4. What makes this meaningfully different from competitors?
5. What measurable business outcome is implied or promised?
6. What should the buyer do next?
7. Can Google crawl, understand, and index it cleanly?

## Kaizen IQ Copy Standard

Avoid:

- Generic marketing agency language.
- Unsupported AI claims.
- Keyword stuffing.
- Vague transformation language.
- Weak CTAs.
- Long copy that does not increase trust or action.

Prefer:

- Executive clarity.
- Mechanism-based claims.
- Strategic specificity.
- Data and AI as operating leverage.
- Responsible implementation.
- Measurable business outcomes.
- Strong diagnostic CTAs.

## Operating Principle

The SEO specialist builds. The researcher grounds the market. The competitor agent maps the battlefield. The trust agent strengthens credibility. The conversion strategist creates revenue intent. The red team applies pressure. The brand editor makes it unmistakably Kaizen IQ. The technical architect ensures the implementation is clean. The analytics agent turns results into compounding advantage.
